zm.infectati.ro (2012-2015)

Acest pachet contine: serverul CS 1.6 mod zombie denumit zm.infectati.ro + baza de date MySQL + panoul web ICP + site-ul de prezentare

Pentru instalare:
1. Incarcati dosarul cstrike pe un server cs linux versiune mai veche de 2015.
2. Creati o baza de date MySQL si importati fisierul sql
3. Modificati sursa plugins-urilor din dosarul scripting numite: admin_zm.sma , zp_sub_bank_sql.sma , zm_vip_sql si efectuati conectarea la baza de date create anterior. Compilati noile plugins-urile si le incarcati pe server.
4. Incarcati panoul web ICP pe un webhost cu versiune php 5.6 si efectuati conectarea la baza de date in fisierul "connect.php".

Admin, VIP si amo in banca se acorda doar din panoul web sau direct din baza de date in tabelul denumit "tz_members".

ATENTIE: Contine brese de securitate si bug-uri. NU ofer suport de instalare si configurare.
